Adding new type

You can add new type of resource. Process of adding should start with pressing btn add->New type from menu bar, or by pressing shortcut key combination CTRL+T




That action will open window that looks like this:


You can type in id, name, description and also upload type icon.


By pressing submit button, if every field is filled and fulfils constraints, window will close and new type will be visible.


After this action, new type is capable of performing same actions as other types, like adding new resource
